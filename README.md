# OrangeHRM Automation Framework

## Project Overview
This project is an automation testing framework developed for the OrangeHRM application using Selenium WebDriver, Java, TestNG, and Maven.

The framework follows the Page Object Model (POM) design pattern and supports data-driven testing using Apache POI.

---

## Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven
- Apache POI
- Extent Reports
- Git & GitHub

---

## Framework Features

- Page Object Model (POM)
- Data Driven Testing
- Valid and Invalid Login Testing
- Excel Data Provider
- Cross Browser Testing
- Screenshot Capture
- Extent Reports
- TestNG Listeners
- Logout Functionality

---

## Project Structure

```text
src/test/java      -> Test Cases
src/main/java      -> Page Objects
testData           -> Excel Test Data
reports            -> Extent Reports
screenshots        -> Failure Screenshots
testng.xml         -> TestNG Suite File
```

---

## Test Scenarios Covered

- Valid Login Test
- Invalid Login Test
- Logout Test

---

## How to Run the Project

1. Clone the repository
2. Open project in Eclipse
3. Install Maven dependencies
4. Run testng.xml file
